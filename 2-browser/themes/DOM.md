+ scrollWidth / scrollHeight

# clientWidth
![clientWidth](https://developer.mozilla.org/@api/deki/files/185/=Dimensions-client.png)

clientWidth = width + padding - 垂直滚动条宽度;

不包括border、margin和垂直滚动条



offsetWidth = width + padding + border;
