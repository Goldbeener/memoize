## 性能指标
### Navigation Timing
收集网络请求的生命周期和计时的性能指标
+ DNS lookup
+ Header Size
+ Fetch time 缓存查找响应时间
+ Worker time Service worker时间加上响应时间
+ Total time 请求响应时间，仅网络
+ Download time 仅下载响应时间
+ TTFB Time To First Byte 请求发出后到接受响应首字节所需时间， web页面加载耗时的最大影响因素，占页面总延迟的40%-60%
